# Praxis

Praxis teaches Vim through short, self-contained challenges: Tutorials, Training, and Trials, all inside Neovim.

## Install

Install the `praxis` binary, open Neovim, and run `:Praxis`.

## Quick Start

Open Neovim and run `:Praxis`.

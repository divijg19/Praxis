# Curriculum

Praxis teaches 41 Vim editing challenges across 6 progressive stages. Each stage introduces a new category of technique and builds on the previous one.

| Stage | Challenges | Verify | Prerequisite |
|---|---|---|---|
| Movement | 3 | cursor | — |
| Search | 7 | cursor | Movement |
| Structural Navigation | 10 | cursor | Search |
| Editing | 7 | buffer | Structural Navigation |
| Text Objects | 9 | buffer | Editing |
| Registers | 5 | buffer | Text Objects |
| **Total** | **41** | **20 cursor, 21 buffer** | |

Challenges are organized by pedagogical stage. The CHALLENGES.md catalog uses historical shipping packs — a separate taxonomy from stages. See PROGRESSION.md for details.
